# 1章 事前準備

## 概要

- 環境周りの事前準備をします

## Nodeのインストール
    
- [公式サイト](https://nodejs.org/)からダウンロードしてください
- macの人はbrewでもインストールできます

```bash
brew install node
```

- 以下のコマンドでバージョン情報が表示されればOK

```bash
node -v
npm -v
```

## yarnのインストール

- `yarn`というコマンドラインツールをインストールします

```bash
npm i -g yarn
```

- 以下のコマンドでバージョン情報が表示されればOK

```bash
yarn -v
```
